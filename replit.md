# Estatevita

A static real estate landing page built with HTML, Tailwind CSS (CDN), and Font Awesome icons.

## Project Structure

- `Estatevita.html` — The single-page HTML file containing the full site (navbar, hero, property listings, etc.)

## Running the Project

The app is served by Python's built-in HTTP server on port 5000:

```
python3 -m http.server 5000 --bind 0.0.0.0
```

## Deployment

Configured as a **static** deployment serving the root directory (`.`).
